About

Javascripture aspires to be a simple but powerful tool for rightly dividing the word of truth.

This code is in early stages of development, so there will be plenty of bugs. If you have any requests for features or bugs please let me know.

The project is entirely open source, so if you want to help out by adding features, or fixing things, please fork me on github.

The ultimate aim is to make the application available across all computing platforms and smart phones, which is why it's built using web technologies (that, and I don't know anything else).

Features

  • Lookup verses in the KJV
  • Find and highlight the strongs number
  • Search by single or muliple strongs number
  • Search the hebrew text
  • Search by word or strongs number
  • Find strongs roots
  • Reverse roots (branches)
  • Find rare words
